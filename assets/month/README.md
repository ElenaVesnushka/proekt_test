# Фоновые изображения по месяцам

Положите **JPEG** в эту папку (ширина от ~1600px). Скрипт пробует имена **в таком порядке**:

1. Английское имя месяца (как в задании): `january.jpg` … `december.jpg`
2. Резерв: `01.jpg` … `12.jpg`

Подложка и затемнение для читаемости задаются в `theme.css` (`body::before` + градиент).

| Файл | Месяц | Настроение |
|------|--------|------------|
| `january.jpg` / `01.jpg` | Январь | Зима |
| `february.jpg` / `02.jpg` | Февраль | Зимний уют |
| `march.jpg` / `03.jpg` | Март | Ранняя весна |
| `april.jpg` / `04.jpg` | Апрель | Цветущая весна |
| `may.jpg` / `05.jpg` | Май | Солнечная весна |
| `june.jpg` / `06.jpg` | Июнь | Начало лета |
| `july.jpg` / `07.jpg` | Июль | Яркое лето |
| `august.jpg` / `08.jpg` | Август | Тёплое лето |
| `september.jpg` / `09.jpg` | Сентябрь | Золотая осень |
| `october.jpg` / `10.jpg` | Октябрь | Осень |
| `november.jpg` / `11.jpg` | Ноябрь | Поздняя осень |
| `december.jpg` / `12.jpg` | Декабрь | Зимняя атмосфера |

Если файла нет, остаётся только сезонный градиент без фото.
